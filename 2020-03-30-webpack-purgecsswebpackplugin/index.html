purgecss-webpack-plugin 可傳遞選項

可參考 purgecss-webpack-plugin Options 可傳遞參數列表，以下為常用的參數配置：

  • paths：Array
    指定要由 PurgeCSS 分析的檔案路徑，默認為 null

  • whitelist：Array
    指定不該被刪除的 CSS 樣式 (具體名稱)，默認為 null

  • whitelistPatterns：Array
    指定不該被刪除的 CSS 樣式 (正規表達式)，默認為 null
